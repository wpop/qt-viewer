# Qt Viewer

A lightweight image viewer built with modern C++, Qt Widgets, and CMake.

## Screenshot

![Qt Viewer grayscale demo](docs/images/qt_viewer_grayscale.png)

## Features

- Load and display images
- Zoom in/out
- Pan images
- Fit image to window
- Modern C++ architecture
- Cross-platform design

## Technologies

- C++17
- Qt 6
- CMake
- OpenCV (planned)
- OpenGL (planned)
